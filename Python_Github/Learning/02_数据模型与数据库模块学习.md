# 02 数据模型与数据库模块学习

## 对应类

- `RepoItem`
- `SkillItem`
- `TrendSnapshot`
- `DBManager`

## 类和表的关系

`RepoItem` 对应 `repos` 表，保存仓库名称、链接、描述、语言和 star 数。

`SkillItem` 对应 `skills` 表，保存从仓库描述中提取出来的技术关键词。

`TrendSnapshot` 对应 `trend_snapshot` 表，保存某一天仓库的热度快照，用来做趋势折线图。

## DBManager 的作用

`DBManager` 只负责数据库连接、建表、插入和查询。其他模块不直接写 SQL，这样可以降低数据库逻辑扩散的风险。

## 汇报重点

数据模型类负责描述“数据长什么样”，`DBManager` 负责“数据怎么存进去、怎么取出来”。
